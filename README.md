# react + redux 미세먼지 조회 프로젝트

## URL
- http://weather-dust.s3-website.ap-northeast-2.amazonaws.com/

## Spec
- react
- redux / react-redux / redux-saga
- webpack v2
- ES6 / 7 (async,await)

## Getting Started

```
$ npm install
```

```
// development ( Hot loader )
$ npm start

// production
$ npm run build
```
