## URL
- http://weather-dust.s3-website.ap-northeast-2.amazonaws.com/

## Spec
- react
- redux / react-redux
- jest
- chai/expect
- webpack v2
- ES6 / 7 (async,await)

## Installation

```
$ npm install
```

## Getting Start

```
// development ( Hot loader )
$ npm start

// production
$ npm run build

// test
$ npm run tests
```
